# 机器学习

import LinkCard from '/components/LinkCard';
import CardGrid from '/components/CardGrid';

在这个板块中，将以Kaggle上 [Binary Classification with a Tabular Employee Attrition Dataset](https://www.kaggle.com/competitions/playground-series-s3e3/overview) 项目为例，
介绍机器学习的基础知识和常用算法，包括数据预处理、特征工程、模型调参等内容。

该数据集基于广为流传的 IBM HR Analytics Employee Attrition & Performance 数据集，是关于员工离职情况的二分类问题，目标是预测员工是否会离职。
